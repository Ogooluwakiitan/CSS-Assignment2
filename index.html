So far, I have learnt HTML and it's various Elements and Attributes used to Structure a web page. CSS is used to add different Styles to your website or Web page. CSS attributes includes width, height,padding, border, color, fonts and so on to your websites. CSS is important because it helps in styling your codes and making them beautiful.

Padding is the space between the content element and the element border. Padding has four properties which are Padding-top, Padding-right, padding-bottom and padding-left. It is one of the CSS Box Model.

Margin is also a CSS Box Model and it's properties are used to define the space around elements. It is the outer-most box model, its properties are margin-top, margin-right, margin-bottom and margin-left.

Border is used to set the border on an element. Border properties are border-style, border-color, border-width and border-radius. They are used to specify the color and size of the border of an element.
